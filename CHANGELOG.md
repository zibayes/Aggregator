# Журнал изменений (CHANGELOG)

## Спринт 1 (21.03.26 – 04.04.26)

### 🔍 Анализ и подготовка
- Проведён анализ литературы по методам извлечения именованных сущностей (NER) в археологических текстах, а также по подходам к OCR и парсингу структурированных документов.
- Начата подготовка корпуса текстов для дообучения NER-модели: выполнено аннотирование первичных данных, определён формат разметки.

### 🧪 Тестирование
- Расширена инфраструктура тестирования:
  - Добавлены новые тестовые модули для проверки геообработки (`test_geo_objects_processing.py`, `test_geo_utils.py`) и утилит (`test_utils.py`).
  - Реализованы тесты для представлений (views): `test_batch_processing.py`, `test_datatable_views.py`.
  - Обновлены существующие тесты обработки учётных карт (`test_account_cards_processing.py`) и вспомогательные конфигурации (`conftest.py`).
- Создан комплект тестов для проверки корректности методов обработки координат и геометрических данных.

### ⚙️ Доработка функциональных модулей
- **Обработка учётных карт** (`account_cards_processing.py`):
  - Улучшена логика извлечения атрибутов, подготовлена основа для интеграции OCR.
- **Батч-обработка** (`batch_processing.py`):
  - Оптимизированы процессы массовой обработки отчётов, исправлены мелкие ошибки.
- **Работа с внешними источниками** (`external_sources.py`):
  - Доработана загрузка данных с внешних ресурсов, повышена стабильность при работе с сетевыми запросами.
- **Обработка геообъектов** (`geo_objects_processing.py`):
  - Реализованы вспомогательные функции для работы с геометрическими объектами, улучшена обработка координат.

### 🖥️ Интерфейс
- Обновлён шаблон дашборда батч-обработки (`batch_processing_dashboard.html`) для отображения дополнительной информации о статусе задач, реализована пакетная обработка учётных карт.

### 📦 Результаты спринта
- Сформирован базовый набор тестов, покрывающий ключевые модули обработки.
- Выполнена рефакторинговая подготовка для интеграции NER и OCR-модулей.
- Создана основа для последующего извлечения атрибутивных данных из учётных карт и текстов отчётов.
